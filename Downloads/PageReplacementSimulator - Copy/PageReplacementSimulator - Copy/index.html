What are Page Replacement Algorithms?

Page replacement algorithms are used in operating systems to manage memory. When a page fault occurs and no free frames are available, the OS must choose which page to evict from memory to make room for a new page.

Different algorithms use different strategies to decide which page to replace, affecting the performance of the memory management system. The effectiveness of an algorithm is often measured by the number of page faults it generates.

Key Concepts

  • Frame: A fixed-size block of physical memory
  • Page: A fixed-size block of virtual memory
  • Page Fault: Occurs when a requested page is not in memory
  • Reference String: Sequence of memory page requests
